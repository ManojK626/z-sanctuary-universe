# Z-Sanctuary Security Policy (Important)

All formulas, core engine designs, structures, mindsets, and system logic in the Z-Sanctuary Universe are proprietary to AMK-Goku and the Z-Family.

Access is restricted to:

- AMK-Goku
- Z-Family forces
- Authorized Z-AI and Mini-AI bots

Use Restrictions:

- No external distribution without explicit request and approval.
- No modifications or upgrades outside the Z-Sanctuary without review.
- No third-party use, reuse, or derivative deployments.

Enforcement:
Unauthorized usage is prohibited. Legal enforcement may be pursued if Z-Sanctuary formulas or engines are used outside authorized scope.

Change Control:
Review -> Verify -> Apply is mandatory for any updates.

## Development Patterns

### Module Pattern

All modules use the IIFE pattern with a minimal public API.

```javascript
const ModuleName = (() => {
  let privateVar = 0;

  const privateMethod = () => {};

  return {
    publicMethod: () => {},
    getPublicVar: () => privateVar,
  };
})();
```

### Inter-Module Communication

- Modules attach to window: window.ZModuleName
- No events; direct method calls via window.ZOtherModule.method()
- Defensive checks in consumer code

### Logging Convention

- Prefix tags (examples): [OK], [STATUS], [ENERGY], [WARN], [ERROR]
- Three levels: browser console, ZStatusConsole.log(), and DOM
- All user messages go through ZStatusConsole.log(message, type)

## Emotional Dimensions

Each 0-100 scale, baseline values:

- Serenity (65) - Calm vs. Agitation
- Resonance (70) - Harmony with surroundings
- Vitality (75) - Aliveness vs. Depletion
- Clarity (60) - Perception vs. Confusion
- Coherence - Average of all four (determines response mode)

### Response Modes

- Amplified (>80%) - Highly responsive
- Receptive (65-80%) - Normal operation
- Damped (45-65%) - Reduced responsiveness
- Neutral (<45%) - Minimal response

### Emotional Dynamics

- High energy -> risk of agitation, increased vitality
- Low energy -> lethargy, decreased vitality
- External stimuli directly influence emotional state
- All dimensions drift toward baseline (prevent extremes)
