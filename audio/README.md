# Audio

Place ambient and UI audio assets here.

Conventions:

- Keep filenames descriptive and lowercase: `ambient_forest_loop.mp3`, `ui_click.wav`.
- Prefer short loops for background ambience (2–30s) and lossless or high-quality compressed files for exports.

The core modules do not currently load audio automatically. Use the HUD or a future audio manager to reference files from this folder.
