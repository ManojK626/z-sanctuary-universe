# Living Workspace Executive Summary

**Audience:** executives, business stakeholders, office/education operators.

## Purpose

Z-Sanctuary’s living workspace is an auditable, human-first automation system designed to reduce risk and accelerate calm, compliant action.

## Key Guarantees

- **Harisha Guard:** Every automation action shows the current mental-state score so humans decide when to pause or proceed.
- **Safe Pack enforcement:** Only allowlisted automations run; every attempt logs to Chronicle + Super Ghost for audit.
- **Living heartbeat:** Heartbeat audio, World Pulse, and Z-Colour stay synchronized with system state.
- **Super Ghost reflections + exports:** Weekly narratives summarize the week; exports provide proof of the living system for stakeholders.

## Deliverables

- Insight panel link to this living workspace brief (`docs/Living-Workspace-Brief.md`).
- Harisha overlay badge + tooltips on automation controls and keyboard shortcuts.
- Harisha + zone status panel showing score, zone, reflection summary, and a “Run reflection now” button.
- Exportable JSON snapshot (via the Insight reflection card) for proof-of-life sharing.
