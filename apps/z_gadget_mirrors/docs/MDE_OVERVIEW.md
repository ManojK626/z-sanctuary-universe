# Z-Mirror Decision Engine (MDE) Overview

`Z-Gadget Mirrors` is a deterministic decision-support module for device ecosystem choices.

## Scope

- Compare device profiles with transparent scoring.
- Estimate switching friction, learning hours, cost delta, and sustainability impact.
- Provide explainable recommendations with visible reason codes.

## Guardrails

- No OS emulation.
- No proprietary UI copying.
- No silent weight mutation.
- No hidden pricing source selection.
- Approval required for any real device action.

## Runtime Mode

- Local-first by default.
- Cloud sync is optional and must preserve user ownership/privacy.
