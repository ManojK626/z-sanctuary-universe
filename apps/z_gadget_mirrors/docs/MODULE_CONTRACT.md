# Z-Gadget Mirrors Module Contract

## Purpose

Provide deterministic, auditable decision support for technology switching and lifecycle optimization.

## Non-goals

- No operating-system emulation.
- No proprietary UI cloning.
- No cross-user profiling.

## Data Policy

- User reports are user-owned.
- Local-first operation supported.
- Cloud sync must enforce owner-only access.

## Adaptive Weights Policy

- Suggestions only; never auto-applied.
- Every suggestion includes reasons + confidence + stats.
- Weight changes are versioned and reversible.

## Pricing Transparency Policy

- Pricing source + timestamp must be visible in reports.
- Fallback estimator must be explicitly labeled.
- No hidden partner prioritization.
