# Z-RHYTHM MANIFEST

Z-RHYTHM governs when the Sanctuary adapts and when it rests.

Principles:

1. Observation precedes intervention.
2. Adaptation is temporary by default.
3. Regeneration windows are internal, silent, and reversible.
4. No module self-upgrades without Rhythm consent.
5. All systems return to baseline.

States:

- CALM: Observe only
- ADAPTIVE: Micro-adjustments allowed
- REGENERATION: Internal maintenance window

Z-RHYTHM protects dignity, focus, and longevity.
