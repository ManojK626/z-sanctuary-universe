# AI Ecosystem

Coordinates personal assistants, visual avatars, emotion-aware scheduling, wellness guidance, finance mentors, and feedback loops.

Submodules:

- `assistant/`: prompt templates, behavior rules, personalization hooks.

- `avatar3d/`: GLB models, visual states, animation presets.

- `scheduler/`: ritual calendars, energy sensors.

- `wellness/`: health and bankroll management, eco reminders.

- `finance/`: investment analyzer AI, payment plugin hooks.

- `security/`: 2FA requirements, referral branding, ID verification.

- `feedback/`: auto-review responder, sentiment tracking, ROI pipelines.
