# Learning Spine Overview (Core v1.0 Safe)

Purpose:
Provide a calm, read-only explanation layer on top of the locked core.

Principles:

- Observation before action
- Education without enforcement
- No permissions unlocked by learning
- No scoring or profiling

Components:

- Orientation (Level 0)
- Learning Cards (KAIROCELL, Rhythm, Feeling, Trust, JailCell)
- Notebook links (Related Learning sidebar)
- “Why?” links on Rhythm + Feeling panels
- Circle Learning (read-only packs)

Entry Points:

- `ui/education/orientation.html`
- `ui/education/learning_cards.html`
- Notebook sidebar (learning prompts)
- Trust Portal “Why?” links

Status:
Core v1.0 compatible. Read-only. Safe to expose publicly.
