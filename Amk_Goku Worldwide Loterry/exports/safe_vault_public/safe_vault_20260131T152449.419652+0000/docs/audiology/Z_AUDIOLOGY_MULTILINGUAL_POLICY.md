# Z-Audiology Multilingual Policy (v1.0)

## Principle

Meaning → Tone → Language. If meaning or tone cannot be preserved, Z‑Audiology chooses silence.


## Rules

- Unsupported language → no audio.
- Low pronunciation confidence → no audio.
- Sensitive/legal content → summary text only.
- Audio is a privilege, not an entitlement.
