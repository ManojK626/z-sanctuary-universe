# Z-Audiology UI Policy (v1.0)

## Required Controls

- Enable Audio (OFF by default)
- Quiet Mode (global)
- Preferred Language
- Voice Layer (System / Guardian / Stillness / Child‑Safe)
- Speed (Slow default)

## Defaults

- Audio OFF
- Quiet Mode ON when uncertainty exists
- Summary‑only speech
- System voice only

## First-Time Explanation (optional)

If user opts in, one sentence:  
“Z‑Audiology reads only what you allow. Quiet Mode always wins.”

## Non‑Negotiables

- Quiet Mode supersedes all triggers
- No background listening
- No auto‑play on page load
