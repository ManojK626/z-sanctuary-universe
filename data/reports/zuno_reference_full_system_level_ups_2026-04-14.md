## 2. Super Chat (Z-All-in-One)

**Docs:** [docs/Z-SUPER-CHAT-BLUEPRINT.md](../../docs/Z-SUPER-CHAT-BLUEPRINT.md)

**Runtime**

- `npm run super-chat:readonly` — HTTP on `127.0.0.1:5510` (override `Z_SUPER_CHAT_PORT` / `PORT`).
- Static UI source: `docs/public/super_chat/index.html` (served by server for same-origin API).
- Evidence append-only: `data/reports/z_super_chat_evidence.jsonl`.

**API surface (server)**

- `GET /`, `GET /api/health`, `GET /api/task-hooks`, `GET /api/task-run-capability`, `GET /api/approval-challenge` (when execute enabled).
- `POST /api/query` — cross-root search, `read_only: true`, path deny list (vault-adjacent, `node_modules`, `.git`, etc.).
- `POST /api/task-execute` — **only if** `Z_SUPER_CHAT_ALLOW_TASK_EXECUTE=1`; requires `EXECUTE` + challenge; **only** hooks with `allowServerExecute` and `npmScript` present in root `package.json`. **Never** `super-chat:readonly` or `super-chat:verify:probe` from this process (deadlock / self-probe).

**Manifest:** `data/z_super_chat_task_hooks.json` — includes allowlisted: `super-chat:verify`, `z:bridge:validate`, `z:bridge:test`.

**UI capabilities**

- Web Speech: **Mic** (STT into query), **Read summary** (TTS on last hits).
- Task list + **Prepare run** dialog, **Copy output** on results.

**Verification**

- `npm run super-chat:verify` — static gates + server source checks.
- `npm run super-chat:verify:probe` — live `/api/health` + `/api/query` (health accepts `read_only` **or** `taskExecuteEnabled`).

**VS Code tasks:** `Z: Super Chat (read-only server)`, `Z: Super Chat Phase A verify`.

## 6. Accessibility model (for Zuno monitoring)

**Storage keys (localStorage, per browser)**

- `zDashA11yLargeTargets`, `zDashA11yHighContrast`, `zDashA11yStrongFocus`, `zDashA11yReduceDecor` — value `'1'` / `'0'`.

**Body classes on `body.z-dashboard-organism`**

- `z-dash-a11y-large` — min ~44px targets on primary buttons / top tabs / tool summaries.
- `z-dash-a11y-contrast` — stronger panel borders / text emphasis.
- `z-dash-a11y-focus` — high-visibility `:focus-visible` rings.
- `z-dash-a11y-reduce-decor` — hides `#zLivingPulseWrap` decorative canvas; kills meter breathe animation.

**Interaction with OS:** Does not replace screen readers or OS “reduce motion”; stacks with them where applicable.
