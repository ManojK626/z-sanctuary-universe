# Gentle Trust Mode (Kids / Elders Walkthrough)

## Purpose

Explain trust surfaces in calm language with no data exposure or execution authority.

## Principles

- Read‑only
- Soft language
- Slow pacing
- No file links by default

## Steps

1. Welcome
2. What the system knows (labels only)
3. What is real (no numbers)
4. Rhythm (calm / adaptive / rest)
5. Learning (not punishment)
6. Summary (no action required)

## Safety

- No identifiers
- No downloads
- No commands
