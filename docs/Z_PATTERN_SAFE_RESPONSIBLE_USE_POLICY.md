# Z-Pattern Safe — Responsible Use Policy

Doctrine for **pattern engines**, **prediction games**, **simulators**, and **trust-style scores** originating from inspirations such as **Any Predictions / Z-ANY** (PDF blueprint stays **out-of-band** until rewritten into hub-safe language).

## A. Locked law

```text
Pattern detection ≠ future certainty.
Prediction game ≠ betting instruction.
Simulation ≠ financial advice.
Confidence score ≠ guarantee.
Bankroll math ≠ financial advice.
Game pattern ≠ casino edge.
GREEN ≠ deploy.
BLUE requires AMK.
RED blocks movement.
AMK-Goku owns sacred moves.
```

## B. Green lane (education / literacy / simulation)

Allowed when outputs are labelled **hypothesis**, **historic fit**, **variance illustration**, **drill scaffolding**, **creative structure**, **civic claim sorting**, **dashboard validators**, or **journalling prompts**.

Must include **human-readable disclaimers**: not legal, not medical, not investment, not wagering direction.

## C. Amber / gated lane (YELLOW / BLUE)

- **YELLOW:** unclear scope, incomplete evidence packs, taxonomy drift — fix mapping before exporting.
- **BLUE:** OCR, wallets, biometric identity, marketplaces, live APIs, user vault storage, monetisation sketches — AMK charter + connector gate (typically Z-XBUS) before build.

Gambling-adjacent and bankroll-labelled ideas MUST sit in **BLUE** as **education or simulation posture**, never GREEN as “helpful tipping.”

## D. Forbidden (RED exporter language)

Exports or automation MUST NOT imply:

| Class | Examples |
| ------------------ | -------------------------------------------- |
| Wagering directive | bet on …, stake ..., sure win … |
| Casino promises | beats the wheel, exploits the house … |
| Financial advisory | diversify into …, APR advice … |
| Certainty creep | guarantees, prophecy, deterministic next ... |
| Manipulation | hidden control, covert targeting |

## E. Translating blueprint ideas

| Blueprint idea | Safe translation |
| ------------------- | -------------------------------------------- |
| Cosmic dashboard | AMK dashboards + signal overlays |
| Pattern engine core | Offline simulation validators + hypotheses |
| AI companion | Explainer framed as tutor — no execution |
| Immutable ledger | Validator JSON/MD receipts — not custody |
| Anti-poisoning | Spam / anomaly hygiene for evidence intake |
| Trust score | Evidence-quality band — never social bribery |
