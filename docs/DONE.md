## 🎯 Key Features

### **Emotional Dimensions**

- **Serenity** (0-100) - Calm vs. Agitation
- **Resonance** (0-100) - Harmony with environment
- **Vitality** (0-100) - Aliveness vs. Depletion
- **Clarity** (0-100) - Perception vs. Confusion
- **Coherence** (0-100) - Emotional health (avg of 4)

### **Response Modes**

- **Amplified** (coherence >80%) - Highly responsive
- **Receptive** (65-80%) - Normal operation
- **Damped** (45-65%) - Reduced response
- **Neutral** (<45%) - Minimal response

### **Dynamic Behavior**

- High energy -> Risk of agitation, increased vitality
- Low energy -> Lethargy, decreased vitality
- External stimuli directly influence emotions
- All states drift toward baseline (prevent extremes)

## 🎭 The Emotional System Explained

### Simple Version

The system has 4 emotional states that change based on system energy and external stimuli. Higher emotional coherence = more responsive system. Lower coherence = system is depleted or confused.

### Technical Version

Z-Emotion Filter maintains 4 independent 0-100 dimensions (serenity, resonance, vitality, clarity). Each pulse updates these dimensions based on energy/harmony levels. External stimuli can directly modify emotions. Response mode is determined by coherence (average of 4 dimensions), which affects system responsiveness. All states have natural decay toward baseline to prevent extreme states.
