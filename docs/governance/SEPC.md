# SEPC - Security, Etiquette, Protocol & Compassion

SEPC is a mandatory system-wide gate for Z-Sanctuary actions.

## Purpose

Ensure every action is:

- safe and lawful
- humane and dignified
- explainable and reviewable
- de-escalatory by default

## Non-Negotiable Principles

1. De-escalation first
2. No intimidation, humiliation, or dominance language
3. Human-in-the-loop for escalation
4. Auditability: actions must be traceable
5. Safety over speed

## The Dual-Pass Rule

No action proceeds unless BOTH are satisfied:

- SKK confirms lawfulness and stability
- RKPK confirms compassion and dignity

If either raises concern, the system pauses and reassesses.
