# Z-UCCR — Visual Effects and Themes

**Purpose:** Future **visual language** for the Universal Canvas. Effects interpret **system posture**—they are not hype, not fake intelligence, not casino energy.

---

## 1. Design intent

Effects answer: _How does it feel to stand in Sanctuary right now?_ They do **not** answer: _How cool is our AI?_

---

## 2. Planned motifs (conceptual catalog)

| Motif | Meaning |
| ---------------------------- | ------------------------------------------------------------------------------------------------ |
| **Living constellation map** | Modules and cores as steady stars—not shooting stars implying success/failure speed |
| **Glowing panels** | Soft halo on active focus; STOP uses firm outline not neon seizure |
| **Neon energy borders** | Allowed only under **Storm** aesthetic when operator opts in; muted by calm/night |
| **Moving data rivers** | Optional; **paused** under reduced-motion and plain sacred mode |
| **Calm / nature modes** | Earth tones, botanical undertones—not gamified rarity colors |
| **Storm mode** | High-churn day intensity; still no flash patterns; audio off by default |
| **Night mode** | Low blue; trust colors desaturated for sleep-friendly ops |
| **Plain focus mode** | Flat panels, no particles—document-first |
| **Reduced-motion mode** | Statics only; parity for all statuses |
| **High-contrast mode** | Borders and focus rings prioritized over glow |
| **Guardian aura states** | Subtle rims per guardian lane ([GUARDIAN_PANEL_REQUIREMENTS.md](GUARDIAN_PANEL_REQUIREMENTS.md)) |
| **Trust signal colors** | Green **only** where receipt-backed; amber truthfully shown; hold = restrained red outline |

---

## 3. Prohibited aesthetic moves

- Pulsating **default** loaders implying “AI thinking magic.”
- Confetti/spark explosions on verifier pass/fail.
- Mimicking trademarked franchises’ UI for clout without license.
- Embedding strobing palettes as default onboarding.
