# Z-UCCR — Operator Workflow

**Purpose:** Describe how **AMK-Goku and approved operators** move through the Universal Canvas when it exists—**Commander-first**, calm steps, explicit actions.

**Related:** Commander discipline in HODP; [ETHICS_AND_CONSENT_GATES.md](ETHICS_AND_CONSENT_GATES.md).

---

## 1. Daily rhythm (conceptual)

1. **Open** calm default view (plain or nature mode per operator preference).
2. **Read** System Status + Trust pointers—note amber honestly.
3. **Choose one lane** (one panel focus); avoid parallel storm unless crisis runbook says so.
4. **Act** only through explicit affordances (copy command, open doc, queue draft—never silent execute).
5. **Record** optional operator note in sanctioned receipt flow if policy requires.

---

## 2. Commander alignment

Founder Command / Commander surfaces remain **advisory read layers** until separate approval grants action chrome. Z-UCCR deepens **spatial context**, not authority override.

## 3. Mini-bot routing (doctrine — future wiring)

Each “bot” is a **role contract** for signals and outputs. None auto-merge or auto-publish by this doc.

| Bot | Input signals | Allowed outputs | Escalation triggers | Forbidden actions |
| ---------------- | ----------------------------------------- | ----------------------------------------------------------------------- | ------------------------------------- | --------------------------------------- |
| **Ops** | verify fail, task drift, timeguard | Checklist draft, command copy | repeated CI red on release path | remote exec, NAS admin without gate |
| **Ethics** | policy change request, copy touching harm | Review memo, blocker flag | minors/vulnerable user impact unclear | autonomous ethics PASS stamp |
| **Scribe** | completion events, changelog need | Receipt draft Markdown | contradictory receipts | overwriting signed releases |
| **Product** | registry row change, Sovereign lite diff | briefing note | live billing implication | SKU/price autonomous change |
| **Guardian** | sentinel, STOP, pause rails | WARN chip text, STOP outline | contradictory human override absent | pretending to BE a real guardian person |
| **Reports** | stale report, INDEX gap | regen pointers, staleness badges | Vault leak suspicion in logs | dumping raw sealed vault |
| **Creative** | era theme, Orchestra brief | draft brief link to [Z-CREATOR-ORCHESTRA.md](../Z-CREATOR-ORCHESTRA.md) | public kid-facing lane | auto-post social/stream |
| **Outreach** | NGO/investor packet request | outline only | unreviewed legal claims | mass email send |
| **Safety** | responsible-use pattern in feedback | escalate human + resources pointer | self-harm language | auto account sanction |
| **Architecture** | cross-module coupling diagram need | map draft JSON spec | Overseer conflict | registry truth rewrite without ACN/MTEH |

Escalations always funnel to **humans first**—Canvas shows **routing**, not adjudication verdicts.
