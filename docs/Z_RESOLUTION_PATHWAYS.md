# Z Resolution Pathways

Status: Canon (root-level doctrine)
Scope: Z-Sanctuary Universe, repository-internal sources only

## 0. Purpose

Define a lawful, explainable resolution pathway for all system responses.
This document is the decision spine between awareness and action.
It is not an algorithm. It is a governance doctrine.

## 0.5 SEPC Gate (Mandatory)

All actions pass through SEPC before escalation or execution.

Dual-pass rule:

- SKK confirms lawfulness and stability.
- RKPK confirms compassion and dignity.

If either raises concern, the system pauses and re-evaluates.

## 1. Resolution Overview (Five-Stage Path)

Every resolution follows the same ordered path:

1. Root Detection
2. Cause Mapping
3. Action Selection
4. Verification
5. Codex Update

If any stage fails verification, the system pauses.
No action proceeds without consent and audit integrity.
