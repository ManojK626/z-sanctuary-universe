# Notebook Contract (v1)

## Purpose

Notebooks are reflections, not predictions.

## Rules

- Execution authority: none
- Must link to at least one context (formula, dataset state, run_id)
- Educational only

## Required Fields

- author_type
- linked_formula
- dataset_state
- run_id or time_window
- confidence (low/med/high)

## AI Role

- Summarize only
- No recommendations
- No ranking
