# OCTAVE Founder Financial Control Checklist (v1.0)

## Before Cash Comes In

- [ ] Separate operating account

- [ ] Tooling ownership in writing

- [ ] Refund policy visible

- [ ] Production cap set

## During Preorders

- [ ] Weekly cash snapshot

- [ ] Refund reserve untouched

- [ ] Supplier payments milestone-based

- [ ] No scope creep

## Before Scaling

- [ ] Margin ≥ 60%

- [ ] Defect rate ≤ 0.5%

- [ ] Worker Profit-Share funded

- [ ] IP fragmentation enforced

## Hard Rules

- No equity deals under pressure

- No subscriptions that coerce

- No claims drift

- No single-supplier dependency
