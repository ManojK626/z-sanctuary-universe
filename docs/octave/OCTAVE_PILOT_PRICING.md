# OCTAVE Pilot Pricing (v1.0)

## Pilot Goal

Validate comfort, fit, clarity, and safety language—then improve publicly.

## Pilot Packages (90 Days)

### Starter Pilot — €6,000

- 20 kits

- Onboarding (60 min)

- Review access + Trust Dashboard

- End-of-pilot report

### Standard Pilot — €12,000

- 50 kits

- Onboarding + mid-pilot check-in

- Evidence review & action loop

- Public “What We Changed” entry (if applicable)

### Extended Pilot — €20,000

- 100 kits

- All above + customization notes

- Replacement parts allowance

- Priority iteration slot

## Notes

- Non-medical, opt-in use

- No data resale

- Discounts for education/shelters available
