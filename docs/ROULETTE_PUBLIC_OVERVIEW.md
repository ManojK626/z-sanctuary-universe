# 🎰 Roulette Module Public Overview

The roulette analytics modules in this repository provide multi-table spin tracking, hot/cold heatmaps, pattern statistics, and simulations designed purely for entertainment, learning, and research. They are not intended to make guarantees about casino outcomes.

See `apps/roulette-calculator/README.md` for the full capability summary, responsible-play guardrails, and quick-start instructions.
