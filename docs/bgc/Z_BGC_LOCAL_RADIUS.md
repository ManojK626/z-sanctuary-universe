# Z BGC Local Radius Intelligence (Z-LRI)

Status: Canon  
Scope: Internal-only (Z-Sanctuary)  
Policy: Review -> Verify -> Apply

## Purpose

Z-LRI enables local, low-risk material discovery without surveillance or extraction abuse.
It focuses on proximity, consent, and repair-first opportunity.

## Radius Rules

- Default radius: 1–5 km (walkable)
- Extended radius allowed only for community-validated needs
- No automatic tracking or background collection
- No GPS logging unless explicitly consented

## Safety Rules

- Tag hazardous materials as unsafe
- Require protective notes for sharp, electrical, or chemical items
- Never request or display private household locations

## Output

Z-LRI produces a local material map that is:

- non-commercial
- consent-first
- revision-friendly

Silence after checks means green.
