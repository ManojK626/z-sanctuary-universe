# Z BGC Non-Monetize Rules

Status: Canon  
Scope: Internal-only (Z-Sanctuary)  
Policy: Review -> Verify -> Apply

## Non-Monetize List

The following may not be monetized or used to extract value:

- Emergency aid
- Essential safety devices
- Community-only repair knowledge
- Elder and child support tools
- Public sanitation or water access adaptations

## Why

These are dignity-critical resources.  
Z-Sanctuary treats them as protected community knowledge, not profit assets.

## Enforcement

Any attempt to monetize protected items:

- triggers observation-only mode
- replaces outputs with educational guidance
- flags the request for governance review

Silence after checks means green.
