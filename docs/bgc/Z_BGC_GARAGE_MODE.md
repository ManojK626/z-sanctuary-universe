# Z BGC Garage Mode (Small Garage Path)

Status: Canon  
Scope: Internal-only (Z-Sanctuary)  
Policy: Review -> Verify -> Apply

## Purpose

Garage Mode is the practical builder path for local repair and repurpose.
It prioritizes durability, safety, and community usefulness over scale.

## Repair-First Flow

1. Inspect item (safety + condition)
2. Repair if possible
3. Repurpose if repair is not viable
4. Validate in local environment

## Documentation (Lightweight)

- Photo or sketch
- Short notes (what was done)
- Safety checklist
- Local validator feedback

## Output

Garage Mode logs are informational and do not trigger execution.
They are used for learning, not ranking.

Silence after checks means green.
