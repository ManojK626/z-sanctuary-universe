# Z Magical Canvas PlayKit — policy (ZMC-1)

## What this is

A **small reusable local kit** (`shared/magical-canvas/`) for magical-style backgrounds, gentle motion, optional guardian silhouettes, and an optional sketch surface with **PNG download**. Intended for ÉirMind, AT Princess & Blacky, Z-QUESTRA PlayGarden tones, demo pages, and future memorial/creative surfaces.

## Free enjoyment layer (allowed)

| Allowed | Notes |
| ----------------------------- | ------------------------------------------------------------------ |
| Background aurora / gradients | CSS-driven; slow drift |
| Fairy orbs | Paw-spark style when using `princess_blacky` hints |
| Celtic-style rings | Decorative only |
| Angel / guardian outlines | Symbolic; not certification |
| Sketch + clear + local PNG | Browser-only export |
| Theme presets | `eirmind`, `princess_blacky`, `z_questra`, `sanctuary_gold` |
| Reduced motion | Honors `prefers-reduced-motion` |
| Comfort / photophobia | Soft glow when `safetyMode: comfort_first` or hub low-sensory cues |

## Princess & Blacky preset

**Theme id:** `princess_blacky`
**Title:** Princess & Blacky Magical Garden
**Tones:** rose-gold, moon-blue, soft violet, paw amber (CSS variables on `:root` kit node).
**Sketch hints:** paws, hearts, stars — copy-only suggestions in toolbar.
