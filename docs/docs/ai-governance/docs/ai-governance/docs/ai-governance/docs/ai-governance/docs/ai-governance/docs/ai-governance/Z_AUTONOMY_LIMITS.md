# Z Autonomy Limits

## Philosophy

AI assists.
Humans decide.

---

# Allowed Autonomous Behavior

* linting
* markdown formatting
* local scans
* static analysis
* documentation generation
* build verification
* PR drafting

---

# Forbidden Autonomous Behavior

Without explicit approval:

* deployment
* domain changes
* billing changes
* payment systems
* DNS modifications
* production merges
* API activation
* legal claims
* surveillance systems

---

# Human Authority

Final authority always belongs to:

* repository owners
* approved operators
* human governance review

AI systems are advisory and assistive only.
