# Z-MAOS — Autonomy Levels (L0–L5)

**Purpose:** Shared ladder for MAOS and future supervised tooling.

| Level | Meaning | Allowed |
| ------ | ------------------------------------------- | -------------------------------------------------- |
| **L0** | Read-only observe / report | Status, lint read, structure verify |
| **L1** | Triage / classify | Tag signals, draft triage rows |
| **L2** | Task router | Suggest commands, doc links (**default**) |
| **L3** | Supervised patch planner | Patch plans after AMK approval |
| **L4** | Autonomous editor | Narrow lanes only if charter + guardrails |
| **L5** | Merge / deploy / payment / delete / connect | **Forbidden** unless explicit charter + human gate |

**Defaults:** MAOS scripts operate at **L0–L2** only.

---

## 2. Promotion

Raising default autonomy requires documented charter update + verification log—not a silent config flip.
