# Z-Sanctuary ecosystem biology map (doctrine metaphor)

Supporting **Z-STILLNESS-LEARN-1**. This is a **mental model** for where truth and motion live — not runtime biology or autonomous control.

## Why a “biology” map?

Stillness Mode asks idle assistants to **orient** without touching production: _where roots are_, _what grows_, _what connects branches_, _where law stacks_, _where the border is_. Metaphors help teams remember **scope** and **forbidden lanes**.

## Layers (symbol → system meaning)

| Biology symbol | System meaning |
| -------------------------- | ----------------------------------------------------------------------------------- |
| **Root system** | Repo folders, registries (`data/*.json`), and source-of-truth markdown |
| **Mango tree** | Mature phases, nourished doctrine, human-gated “fruit” (receipts, sealed scope) |
| **Tarzan tree** | Navigation between modules/capsules — routing docs, MOD-DIST, fusion maps |
| **Pyramid** | Law stack: Hierarchy Chief, gates, charters, phase receipts |
| **Quadruple spine** | Cognitive / Data / Build / Oversight coordination rails (doctrine-first in Phase 1) |
| **MAGE formula layer** | Governance vocabulary rollup (Ultra MAGE codex — **not** a live engine) |
| **Replica fabric** | Task-scoped replica **doctrine**; bounded missions; no silent spawn |
| **XBUS border** | External connector membrane — permission class only until chartered |
| **Traffic lights** | RED / BLUE / YELLOW / GREEN advisory movement law |
| **AMK dashboard** | Conscious cockpit — indicators and overlays (**read-only** in these phases) |
| **Zuno / AMK Personal AI** | Summary and guidance surfaces — observation, not sacred execution |

Machine-readable aliases for several of these symbols live under `ecosystem_biology` in `data/z_stillness_learning_registry.json`.

## Stillness takeaway

```text
AI learns where it lives in the organism.
AI learns what it can describe.
AI learns what it must never execute.
AI learns which lane to cite for routing.
AI learns how to report without acting.
```

## Locked reminders

- **Ecosystem biology ≠ runtime control.**
- **Stillness output ≠ execution.**
- **BLUE requires AMK** for policy expansions.
- **RED blocks movement** toward autonomy, deploy, providers, secrets, and live connectors.
