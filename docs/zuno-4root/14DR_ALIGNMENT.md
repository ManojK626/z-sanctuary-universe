# 14 DRP alignment (ZUNO-4ROOT)

## Purpose

Anchor ecosystem behaviour to the **14 DRP Protocols** as **human-operated** compassion discipline — not an automated ethics engine claiming certainty.

## Practice

- Governance docs and checklists reference DRP where decisions affect people, animals, vulnerable groups, or public trust.
- AI outputs **propose** alignment notes; humans **approve** posture on serious releases.

## Relation to dashboard

The SKK/RKPK dashboard **signals** that DRP matters (labels, links). It does not compute a moral score or auto-block merges without configured hub gates assigned to humans.
