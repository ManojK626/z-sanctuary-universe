# Z PlayGarden — Safety Policy

## Principles

Z PlayGarden is **fun, proud, colorful, and educational** while staying **local-only** and aligned with **14 DRP** discipline: dignity, consent-aware UX, no harm, no dark patterns.

## Disallowed mechanics

| Disallowed | Notes |
| ------------------------ | --------------------------------------------------------------------- |
| Gambling | No stakes, odds, or wagering |
| Loot boxes | No random paid rewards |
| Paid rewards | No IAP tied to badges |
| Public competitive shame | No stranger leaderboards for kids |
| Pressure streaks | No addictive “lose your streak” loops |
| Rapid flashing | No seizure-risk strobing; kaleidoscope drift is **slow**; no strobing |
| Prediction framing | No “foretell”, “sure win”, fortune, or outcome guarantees |
| Stranger chat | No social chat backend |
| Live hub coupling | No automatic Z-Sanctuary bridge or telemetry |

## Uncertainty Kaleidoscope

| Rule | Notes |
| ----------------- | ------------------------------------------------------------ |
| Illustrative only | Teaches randomness vs pattern — **not** predictive |
| No stakes | “New gentle mix” is a visual reseed, not gambling |
| Motion | Paused under reduced motion / photophobia / OS accessibility |

## Receipt Poster

| Rule | Notes |
| ----------------- | ------------------------------------------------------------ |
| Local export only | SVG/PNG download on device — no upload pipeline added |
| Not a score | Copy states memento is not a prize, leaderboard, or forecast |
| Honest scope | Z-Zebras / Guardian lines stay roadmap-safe |

## Allowed patterns

| Allowed | Notes |
| ---------------- | ---------------------------------------------------------------------- |
| Creativity | Badge Studio previews, calm visuals |
| Kindness framing | Copy emphasizes care and safety |
| Local pride | Non-monetary badges, local-only |
| Learning hooks | Route explorer ties to real panels |
| Accessibility | Respects Comfort bar motion + OS reduced motion; photophobia dims glow |

## Motion

| Condition | Behavior |
| ---------------------------------- | --------------------------------------------------------- |
| `prefers-reduced-motion: reduce` | Orbit animation off |
| Comfort motion = reduced | Orbit animation off |
| Photophobia brightness | Orbit animation off; softer planets/stars |
| OS reduced-motion flag on `<html>` | Orbit animation off |
| Same gates apply | Kaleidoscope slow drift **off** under the same conditions |
