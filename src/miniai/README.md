# Mini-AI Layer

This directory contains the scaffolding for Z-Sanctuary's Mini-AI engine, including base bot classes, messaging schema, and arbitration logic.

## Components

- `base_bot.py`: Python base class for mini-bots
- `redis_schema.txt`: Redis messaging schema (placeholder)
- `arbitration_logic.txt`: Supreme Core arbitration logic (placeholder)

## Example Usage

To run a sample Mini-AI bot:

```bash
cd src/miniai
python run_bots.py
```

This will instantiate JoyBot, send it joy events, and print its state.
