# UI flow (Z-SME)

## Placement

**Z Toolset Comfort Zone** → **Music (SME)** panel (Z-QUESTRA).

## Blocks

1. **Disclaimers** — not medical; no autoplay; local observer log only.
2. **Context** — `training` / `rest` / `reflection`.
3. **Mode pills** — ALIGNMENT | POWER | JOURNEY (explicit selection overrides inference).
4. **Now playing card** — title + expandable “Why this track?”
5. **Playlist** — up to 6 rows
6. **Feedback** — 👍 👎 + quick tags (“too intense”, “too slow”)
7. **Next** — “Training intent” / “Reflection note” buttons → **append memory only** + copy-friendly text for Amk-Goku when hub exists

## Playback

v1: explicit **Play** control is reserved for when licensed URLs are configured. Default copy explains that playback is user-started only.
