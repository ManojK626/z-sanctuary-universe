# Z-NUMEN — Cognitive Geometry Pattern Literacy Engine

## Purpose

Z-NUMEN is an **educational pattern-literacy layer**: it teaches how people notice repetition, numeric salience, triangular or pyramid-shaped symbolic structures, and attention flow **without** asserting hidden agendas, supernatural proof, medical meaning, or certain prediction.

## Companion to Z-Logical Brains

| System | Role |
| --- | --- |
| **Z-Logical Brains** | Learning pathway / cognitive growth map — stages, categories, reflection, visual progression |
| **Z-NUMEN** | Pattern literacy / perception-awareness geometry — repetition, triangles, saturation bands, Zero Core grounding |

Integration is **reference-only** for Phase 1: shared doctrine and registries, not a merged runtime engine.

Formal companion map (**Z-NUMEN-LB-1**): `docs/Z_NUMEN_LOGICAL_BRAINS_COMPANION_MAP.md` and `data/z_numen_logical_brains_companion_map.json` (`mode: reference_only_companion_mapping`, `runtime_coupling: false`).

## What Z-NUMEN teaches

- **Repetition** — why repeated motifs feel “loud” to perception
- **Emotional pressure** — how arousing material changes salience (literacy framing only)
- **Attention flow** — where the eye/mind anchors in a sequence
- **Pattern intensity** — a local, exercise-only score for teaching
- **Zero-core grounding** — return to neutral observation (“observer point”)
- **Triangle / pyramid symbolic structure** — minimum relations (3 points), faces/edges counts as **pedagogy**, not proof of external control
