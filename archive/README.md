# Archive

This folder stores recorded Chronicle artifacts and exported media.

Conventions:

- `.zcrystal` — exported chronicle objects (JSON-like). Currently generated in-memory; saved files go here.
- `*.mp4`, `*.wav` — recorded media and exports.

Note: `.gitignore` already excludes `archive/*.mp4` and `archive/*.zcrystal`.

If you add persistence, write exports to this folder.
